Instructions

  • Click - Add Node
  • Shift-Click/Drag - Create Edge
  • Drag - Move Node/Change Edge Weight
  • Shift-Click - Restore Edge Weight to Default
  • Command-Click - Delete Node/Delete Edge

Make your graph, then click "Run Search" to view the algorithm's output. You can vary the speed with the speed control.
